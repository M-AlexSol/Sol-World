# Chá de Fralda do Apollo

Este projeto é uma página web dedicada ao "Chá de Fralda do Apollo". A página permite que os convidados escolham presentes para o bebê e inclui um timer de contagem regressiva para o evento.

## Estrutura do Projeto

O projeto contém os seguintes arquivos:

- **cha-fralda-apollo.html**: Este arquivo contém a estrutura HTML principal da página, incluindo o layout, estilos e uma seção para o timer de contagem regressiva.
  
- **css/styles.css**: Este arquivo contém os estilos CSS para o projeto, definindo a aparência visual dos elementos HTML, incluindo layout, cores, fontes e design responsivo.

- **js/countdown.js**: Este arquivo contém o código JavaScript para o timer de contagem regressiva. Ele calcula o tempo restante desde a data de início (09-11) até a data de término (30-12) e atualiza a exibição do timer na página.

## Como Usar

1. **Clone o repositório**:
   ```
   git clone <URL do repositório>
   ```

2. **Abra o arquivo `cha-fralda-apollo.html` em um navegador** para visualizar a página.

3. **Personalize os presentes** na lista conforme necessário, editando o conteúdo no arquivo HTML.

4. **Ajuste os estilos** no arquivo `css/styles.css` para modificar a aparência da página.

5. **Modifique o timer** no arquivo `js/countdown.js` se precisar alterar as datas ou o formato de exibição.

## Recursos

- Layout responsivo para dispositivos móveis.
- Timer de contagem regressiva que acompanha o usuário ao rolar a página.
- Links para compra de presentes diretamente de plataformas online.
